# Cornerstone Home & Outdoor

Proyecto estatico basado en la pagina original de `/Users/valeria/Documents/pag/index.html`.

## Estructura

- `index.html`: pagina principal
- `styles/main.css`: estilos globales
- `scripts/main.js`: JS base
- `thank-you.html`: pagina de confirmacion del formulario

## Como abrirlo

Abre `index.html` directamente en el navegador o sirve esta carpeta con cualquier servidor estatico.

## Siguiente paso sugerido

Reemplazar las imagenes temporales de Unsplash por contenido real de la marca y conectar el formulario a un backend o servicio de formularios.
